== Installation ==

### Installing Accessible ComboBox for Gravity Forms Within WordPress
1. Visit the plugins page within your dashboard and select ‘Add New’.
2. Search for ‘Accessible ComboBox for Gravity Forms’;
3. On the Accessible ComboBox for Gravity Forms plugin, click the 'Install' button.
4. Activate Accessible ComboBox for Gravity Forms on your Plugins page.

### Installing Accessible ComboBox for Gravity Forms Manually

1. Upload the unzipped ‘accessible-combobox-gravity-forms’ folder to the /wp-content/plugins/ directory on your website via FTP.
2. Activate Accessible ComboBox for Gravity Forms on your Plugins page.


== Frequently Asked Questions ==

= Does this work with other form plugins? =

No. This plugin only works with Gravity Forms. Gravity Forms is a premium form plugin. You can <a href="https://www.gravityforms.com/">learn more about Gravity Forms here</a>.
